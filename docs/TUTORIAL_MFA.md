# Tutorial de Autenticação de Dois Fatores (MFA) - Honda Go

A Autenticação de Dois Fatores (MFA) adiciona uma camada extra de segurança à sua conta administrativa. Além da senha, você precisará de um código gerado pelo seu celular para entrar no sistema.

---

## 1. Aplicativos Recomendados

Você precisará instalar um dos seguintes aplicativos no seu celular:

*   **Google Authenticator** (Disponível para Android e iOS)
*   **Microsoft Authenticator** (Disponível para Android e iOS)
*   **Authy**

---

## 2. Como Configurar (Primeira Vez)

1.  **Acesse o Menu**: Clique no seu nome no canto superior direito e selecione **"Segurança (MFA)"**.
2.  **Inicie o Setup**: No modal que abrir, clique em **"Configurar MFA Agora"**.
3.  **Vincule seu Celular**:
    *   Abra o aplicativo de autenticação no seu celular.
    *   Toque no botão de adicionar conta (geralmente um símbolo de **+**).
    *   Escolha **"Ler código QR"** e aponte a câmera para o QR Code exibido na tela do computador.
    *   *Dica: Se estiver usando o Honda Go no próprio celular, use a opção "Copiar Chave" e cole-a manualmente no aplicativo.*
4.  **Confirme a Ativação**: O aplicativo começará a gerar um código de 6 dígitos para o "HondaGo". Digite esse código no campo solicitado no sistema e clique em **"Ativar Autenticação"**.

Pronto! Sua conta agora está protegida.

---

## 3. Como Fazer Login com MFA

Após ativar o MFA, o fluxo de login mudará ligeiramente:

1.  **Senha**: Digite seu usuário e senha normalmente e clique em "Entrar".
2.  **Desafio MFA**: O sistema solicitará o código de segurança.
3.  **Abra o App**: Abra o Google/Microsoft Authenticator no seu celular, localize o código de 6 dígitos do **HondaGo**.
4.  **Digite o Código**: Digite os números no sistema e clique em **"Verificar Código"**.

---

## 4. Dicas de Segurança e Recuperação

*   **Não apague o app**: Se você apagar o aplicativo ou perder o celular sem ter feito backup, perderá o acesso à conta.
*   **Acesso bloqueado?**: Se por algum motivo você não conseguir gerar o código, entre em contato com o administrador do sistema para que ele realize o "Reset de MFA" do seu usuário.
*   **Chave Manual**: Guarde a chave secreta (alfanumérica) em um local seguro; ela permite restaurar o acesso se você trocar de celular.
